# How to get started

## Clone the repo

```sh
git clone https://github.com/toburger/tic-tac-toe--skeleton.git
```

## Either kickstart with `npm`

```sh
cd tic-tac-toe--skeleton
npm install
code . # open with the editor of your choice
npm start
```

## Otherwise kickstart with `yarn`

```sh
cd tic-tac-toe--skeleton
yarn
code . # open with the editor of your choice
yarn start
```
